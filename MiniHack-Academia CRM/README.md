# Academia CRM

Enterprise Sales & Outreach Platform for Managing Academic Initiatives.

## Deployment on Render

This repository has been configured to be "Render-friendly" with an automatic [render.yaml](file:///c:/Users/rahul/Downloads/academia-crm-ready/render.yaml) file.

### How to Deploy:

1. **Push this code to GitHub**: Create a repository on GitHub and push these files there.
2. **Go to Render**: Log into your [Render Dashboard](https://dashboard.render.com).
3. **New Blueprint Instance**:
   - Click **New +** in the top right.
   - Select **Blueprint**.
   - Connect your GitHub repository.
4. **Deploy**: Render will automatically detect the `render.yaml` file, configure the Static Site service pointing to the `./MiniHack-Academia CRM` directory, and deploy the app!
